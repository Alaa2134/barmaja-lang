# تصميم لغة بَرمَجة العربية

تهدف هذه الوثيقة إلى تحديد الكلمات المفتاحية والقواعد النحوية للغة البرمجة العربية "بَرمَجة"، والتي ستكون بديلاً للغة JavaScript مع الحفاظ على نفس الوظائف الأساسية. سيتم تصميم اللغة لتكون سهلة الفهم للمتحدثين باللغة العربية، مع التركيز على الوضوح والبساطة.

## 1. الكلمات المفتاحية الأساسية

سيتم استبدال الكلمات المفتاحية الشائعة في JavaScript بنظائرها العربية لإنشاء تجربة برمجة طبيعية باللغة العربية.

| كلمة JavaScript المفتاحية | كلمة بَرمَجة المفتاحية | الوصف |
| :----------------------- | :--------------------- | :---- |
| `var`                    | `متغير`                 | تعريف متغير ذو نطاق وظيفي. |
| `let`                    | `دع`                   | تعريف متغير ذو نطاق كتلي. |
| `const`                  | `ثابت`                  | تعريف ثابت ذو نطاق كتلي. |
| `function`               | `دالة`                  | تعريف دالة. |
| `if`                     | `إذا`                   | تنفيذ كتلة تعليمات برمجية بشكل شرطي. |
| `else`                   | `وإلا`                  | تنفيذ كتلة تعليمات برمجية إذا كان الشرط السابق خاطئًا. |
| `for`                    | `لكل`                   | حلقة تكرارية. |
| `while`                  | `طالما`                 | حلقة تكرارية تستمر طالما كان الشرط صحيحًا. |
| `return`                 | `أرجع`                  | إرجاع قيمة من دالة. |
| `class`                  | `صنف`                   | تعريف صنف (كائن). |
| `this`                   | `هذا`                   | يشير إلى الكائن الحالي. |
| `new`                    | `جديد`                  | إنشاء كائن جديد. |
| `import`                 | `استورد`                | استيراد وحدات (modules). |
| `export`                 | `صدر`                   | تصدير وحدات (modules). |
| `true`                   | `صحيح`                  | القيمة المنطقية صحيح. |
| `false`                  | `خطأ`                   | القيمة المنطقية خطأ. |
| `null`                   | `فارغ`                  | القيمة الخالية. |
| `undefined`              | `غير_معرف`              | القيمة غير المعرفة. |

## 2. المعاملات (Operators)

سيتم استخدام المعاملات القياسية في JavaScript، ولكن يمكن النظر في بدائل عربية لبعضها في مراحل لاحقة إذا لزم الأمر.

| نوع المعامل | المعاملات | الوصف |
| :---------- | :--------- | :---- |
| حسابية      | `+`, `-`, `*`, `/`, `%`, `**` | الجمع، الطرح، الضرب، القسمة، باقي القسمة، الأس. |
| مقارنة      | `==`, `!=`, `===`, `!==`, `>`, `<`, `>=`, `<=` | المساواة، عدم المساواة، المساواة الصارمة، عدم المساواة الصارمة، أكبر من، أصغر من، أكبر من أو يساوي، أصغر من أو يساوي. |
| منطقية      | `&&` (و), `||` (أو), `!` (ليس) | و المنطقية، أو المنطقية، ليس المنطقية. |
| إسناد       | `=`, `+=`, `-=`, `*=`، `/=`, `%=`, `**=` | إسناد قيمة، إسناد مع جمع، إسناد مع طرح، وهكذا. |

## 3. بنية الجملة (Syntax Structure)

ستتبع بنية الجملة في "بَرمَجة" بشكل كبير بنية JavaScript لتسهيل عملية التحويل (transpilation).

*   **كتل التعليمات البرمجية**: ستُستخدم الأقواس المعقوفة `{}` لتحديد كتل التعليمات البرمجية (مثل كتل `إذا`، `دالة`، `لكل`).
*   **العبارات**: ستُنتهي كل عبارة بفاصلة منقوطة `;`.
*   **الأقواس**: ستُستخدم الأقواس `()` للدوال والشروط.
*   **التعليقات**: 
    *   تعليق سطر واحد: `// هذا تعليق سطر واحد`
    *   تعليق متعدد الأسطر: `/* هذا تعليق
        متعدد الأسطر */`

## 4. أنواع البيانات (Data Types)

ستدعم "بَرمَجة" نفس أنواع البيانات الأساسية في JavaScript:

*   **أرقام**: `5`, `3.14`
*   **نصوص**: `'مرحباً بالعالم'`, `"بَرمَجة"`
*   **منطقية**: `صحيح`, `خطأ`
*   **مصفوفات**: `[1, 2, 3]`
*   **كائنات**: `{ مفتاح: قيمة }`

## 5. مثال على الكود

```javascript
// مثال على دالة ترحب بالمستخدم
دالة ترحيب(اسم) {
    إذا (اسم === غير_معرف || اسم === فارغ) {
        أرجع 'مرحباً أيها الزائر!';
    } وإلا {
        أرجع 'مرحباً يا ' + اسم + '!';
    }
}

دع رسالة = ترحيب('علاء');
صدر رسالة;

صنف شخص {
    ثابت نوع = 'إنسان';
    دع اسم;
    دع عمر;

    جديد(اسم, عمر) {
        هذا.اسم = اسم;
        هذا.عمر = عمر;
    }

    دالة قل_مرحباً() {
        أرجع 'أهلاً، أنا ' + هذا.اسم + ' وعمري ' + هذا.عمر + ' سنة.';
    }
}

دع علاء = جديد شخص('علاء', 30);
استورد { قل_مرحباً } من علاء;

// مثال على حلقة لكل
لكل (دع i = 0; i < 5; i++) {
    // قم بعمل شيء هنا
}

// مثال على حلقة طالما
دع عداد = 0;
طالما (عداد < 3) {
    عداد++;
}
```

هذا التصميم الأولي يضع الأساس للغة "بَرمَجة". الخطوة التالية ستكون بناء المترجم (Transpiler) الذي سيحول هذا الكود العربي إلى JavaScript قياسي.
